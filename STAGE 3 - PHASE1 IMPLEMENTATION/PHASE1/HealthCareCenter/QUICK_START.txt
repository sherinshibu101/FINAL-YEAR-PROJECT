## ğŸ› Troubleshooting

**Q: "Port 3000 already in use"**
A: Kill existing process: `Get-Process node | Stop-Process -Force`

**Q: "Cannot connect to database"**
A: Ensure Docker container is running: `docker ps`

**Q: "Login fails with '404'"**
A: Check IAM server is running on port 4000

**Q: "CORS error"**
A: Check `vite.config.js` proxy configuration

**Q: "Rate limit exceeded"**
A: Restart IAM server to reset rate limiter
